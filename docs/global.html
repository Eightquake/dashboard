problem

Error handler script, that any code can emit an event on. When an event error, warn, or info is emitted a popup will be shown to the user. All scripts should use this to communicate with the user, instead of using the console or similar.

See:
  • module:problemHandler for more information.

schedule

Scheduler module, works great and handles all of the scheduling. When a plugin needs to schedule something it should use this.
