Module: fillGrid

Fills the grid by creating new elements for every detail, and then calling every plugin the detail relies on with references to the grid-item and detail.

Methods

(inner) fillGrid(loaded_details, loaded_plugins) → {Promise}

The only function in this module. It fills the div.grid with a new div.grid-item for every detail that is loaded and then it calls all of the plugins that the detail needs. It is exported and when called on it returns a promise that runs the code and resolves when it's done.
Parameters:
Name Type Description
loaded_details Map The Map of all of the loaded details where key is the name of the detail and value is the actual detail
loaded_plugins Map The Map of all of the loaded plugins where key is the name of the plugin and value is what the plugin exports
Source:
  • scripts/renderer/fillGrid.js, line 16
Returns:
A promise that will resolve once all of the grid-items have been created and all of the details needed plugins have been called.
